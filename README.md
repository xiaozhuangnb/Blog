# Blog
基于若依框架进行二次开发的一个博客项目
优化点：本地上传改

## 系统环境需求

- JDK 17
- MySQL  5.7
- Maven 3.6
- Node 17

## 技术选型

1、系统环境

- Java EE 8
- Servlet 3.0
- Apache Maven 3.6

2、主框架

- Spring Boot 2.2.x
- Spring Framework 5.2.x
- Spring Security 5.2.x

3、持久层

- Apache MyBatis 3.5
- Hibernate Validation 6.0
- Alibaba Druid 1.2

4、视图层

- Vue 2.6
- Axios 0.21
- Element 2.15
